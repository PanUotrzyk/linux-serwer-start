# 🐧 Projekt: Konfiguracja serwera Linux (Ubuntu)

## 🎯 Cel
Podstawowa konfiguracja nowego serwera Ubuntu:
- Instalacja Nginx
- Utworzenie użytkownika z dostępem SSH
- Skonfigurowanie firewalla (UFW)
- Przykładowe skrypty do automatyzacji

## 🛠️ Technologie
- Linux (Ubuntu 22.04)
- Terminal / Bash

## 📂 Struktura
- `konfiguracja/` – skrypty i ustawienia
- `notatki.md` – krótkie wyjaśnienia do nauki
